# 🐍 Snake Game 

A fun and interactive Snake Game built using **HTML, CSS, and JavaScript**.  
Control the snake, eat food, grow longer, and try to achieve the highest score without hitting the walls or yourself.

---

## ✨ Features 

- 🎮 Smooth snake movement
- 🍎 Random food generation
- 📈 Score tracking
- 💀 Game over detection
- ⚡ Simple and responsive UI
- ⌨️ Keyboard controls

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 🚀 How to Run

### 1️⃣ Clone the repository

```bash
git clone https://github.com/rewati1616/snake-game.git
```

### 2️⃣ Open the project folder 

```bash
cd snake-game
```

### 3️⃣ Run the project

Open `index.html` in your browser.
